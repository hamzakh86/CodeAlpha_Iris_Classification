## 🎯 Objectif

- Scikit-learn
- Pandas
- Matplotlib & Seaborn

- `iris_classification.ipynb` : Version Jupyter Notebook.
- `requirements.txt` : Liste des dépendances Python.

- Classification Report et Matrice de confusion inclus.
